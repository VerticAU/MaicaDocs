# Create a Shift

## How do I create an Shift?

A quick and efficient way to create [Shift](../../getting-started/maica-key-concepts/shift.md) is directly through the the Maica [Planner](../../the-planner/planner-overview.md). Maica has an interactive user mechanism where you can simply drag and release your mouse on the [Planner](../../the-planner/planner-overview.md) in order to prompt the creation of an [Shift](../../getting-started/maica-key-concepts/shift.md).&#x20;

{% hint style="info" %}
Creating an Shift is dependant on what [View](../../the-planner/planner-views-and-modes/) you are in within the Planner. You need to be in one of `Roster` or `Shift` View in order to create a Shift, otherwise you will [create an Appointment](../../appointments/create-an-appointment/).&#x20;
{% endhint %}

You can also create a Shift through the Planner Quick Action. To learn more, click [here](../../the-planner/planner-actions/create-new-appointment.md).&#x20;
{% endhint %}

## Pre-populated Fields

When creating a [Shift](../../getting-started/maica-key-concepts/shift.md), a number of fields will be pre-populated. These fields depend on the [Planner View](../../the-planner/planner-views-and-modes/) the user is in when the [Shift](../../getting-started/maica-key-concepts/shift.md) is created. The pre-populated fields for each view are outlined in the table below:

## Shift Stages

After the new [Shift](../../getting-started/maica-key-concepts/shift.md) is created, a few stages must be completed before the [Shift](../../getting-started/maica-key-concepts/shift.md) can be confirmed.&#x20;

{% hint style="info" %}
The available sections during the creation or management of a [Shift](../../getting-started/maica-key-concepts/shift.md) are determined by the Maica Settings. For further information, see the [Administration Guide](https://app.gitbook.com/s/9selzjEx6KX7RYEawAVr/settings/service-management).
{% endhint %}

Below is a list of these stages:

1. [Basic Details](basic-details.md)&#x20;
2. [Location](location.md)&#x20;
3. [Schedule](schedule.md)&#x20;
4. [Additional Details](additional-details.md)&#x20;
5. [Shift Instructions](shift-instructions.md)&#x20;
6. [Custom Fields](custom-fields.md)&#x20;
7. [Participant Notes](participant-notes.md)&#x20;
8. [Summary](summary.md)&#x20;

To dive deeper into each stage, please visit the specific pages for detailed information on every stage.&#x20;
