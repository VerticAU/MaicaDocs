# Billable Participant Note

## How do I create a new Billable Participant Note from the Planner?&#x20;

There are multiple ways to create a new Billable Participant Note in **Maica**, this article will show you how to use the Quick Action in the Planner.

{% hint style="info" %}
You can also create a Billable Participant Note through the [Maica Quick Actions](../../participants/global-actions/new-billable-participant-note.md).&#x20;
{% endhint %}

Simply begin by clicking the `New Billable Participant Note` button, as shown below.&#x20;

<figure><img src="../../.gitbook/assets/new billable notes.png" alt=""><figcaption></figcaption></figure>

Once done, the `New Billable Participant Note` module will display.&#x20;

Once the module is displayed, please complete your Note by following the instructions described [here](../../participants/billable-participant-notes/).
